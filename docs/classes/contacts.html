search

  • search(username?: undefined | string, address?: undefined | string, options?: QueryOptions): RunningEvent
    • Defined in modules/contacts.ts:82

    Searches locally and on the network for contacts by username, peer id, or address

    example

    const { emitter, source } = textile.contacts.search(undefined, undefined, {wait: 5}) setTimeout(() => source.cancel(), 1000) // cancel after 1 second emitter.on('textile.contacts.found', console.log) emitter.on('textile.contacts.done', console.log) })

    Parameters

    • Optional username: undefined | string

      Search by username string

    • Optional address: undefined | string

      Search by account address string

    • Optional options: QueryOptions

      Additional options to control the query

    Returns RunningEvent

    Event emitter with found, done, error events on textile.contacts.

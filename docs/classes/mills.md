###  run

▸ **run**(name: *`string`*, options: *[KeyValue](../interfaces/keyvalue.md)*, payload: *[KeyValue](../interfaces/keyvalue.md)*, headers: *[KeyValue](../interfaces/keyvalue.md)*): `Promise`<`any`>

Run a mill over a given payload

Currently supports:

*   `/blob` - Process raw data blobs
*   `/image/exif` - Extract EXIF data from an image
*   `/image/resize` - Resize an image
*   `/json` - Process (and validate according to schema.org definition) input JSON data
*   `/schema` - Validate, add, and pin a new JSON-based Schema

**Parameters:**

| Name | Type | Description |
| ------ | ------ | ------ |
| name | `string` |  Name of the mill. (Relative uri). See above description for details |
| options | [KeyValue](../interfaces/keyvalue.md) |  Schema options for the mill |
| payload | [KeyValue](../interfaces/keyvalue.md) |  A multi-part form containing the payload |
| headers | [KeyValue](../interfaces/keyvalue.md) |

**Returns:** `Promise`<`any`>

(async) add(name, optionsopt)

Adds and joins a new thread
Parameters:
Name Type Attributes Description
name string The name of the new thread
options Object <optional>
 Additional options to send as headers
Properties
Name Type Attributes Description
key string <optional>
 A locally unique key used by an app to identify this thread on recovery
type string <optional>
 The type of thread, must be one of 'private' (default), 'readonly', 'public', or 'open'
sharing string <optional>
 The sharing style of thread, must be one of 'notshared' (default), 'inviteonly', or 'shared'
members Array.<string> <optional>
 An array of contact addresses. When supplied, the thread will not allow additional peers, useful for 1-1 chat/file sharing or private groups.
schema string <optional>
 Schema ID for the new thread
Source:
  • modules/threads.js, line 37
Example
await textile.thread.add("MyMedia", {
  schema: mediaSchema.id,
  type: "open",
  sharing: "shared"
})
